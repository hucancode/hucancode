Binary Indexed Tree (Fenwick Tree)

Posted 02/07/2023

What is a Binary Indexed Tree

Binary Indexed Tree is a data structure that can store array AAA of length nnn with following property

  • Calculates the value of function fff in the given range [l,r][l, r][l,r] in O(log(n))O(log(n))O(log(n)) time. There are some restriction on which kind of function fff you can use, for example sum function works, but min/max function don’t.
  • Updates the value of an element of AAA in O(log(n))O(log(n))O(log(n)) time
  • Requires only O(n)O(n)O(n) spaces, which is the input spaces

Why do we need it

Naive calculation of fff in the range [l,r][l,r][l,r] usually cost O(n)O(n)O(n). Imagine you have 10510^5105 queries, each of them traverse through an array length 10510^5105, you would end up having 10 billions traversal and TLE.

Definition of ggg

The computation of g(i)g(i)g(i) is defined using the following simple operation: we replace all trailing 1 bits in the binary representation of iii with 0 bits.

In other words, if the least significant digit of iii in binary is 0, then g(i)=ig(i)=ig(i)=i. And otherwise the least significant digit is a 1, and we take this 1 and all other trailing 1s and flip them.

There exists a simple implementation of ggg using bitwise operator as follow

g(i)=i&(i+1)g(i) = i\&(i+1)g(i)=i&(i+1)

C++ implementation

BIT 1D

struct FenwickTree {
    vector<int> bit;  // binary indexed tree
    int n;

    FenwickTree(int n) {
        this->n = n;
        bit.assign(n, 0);
    }

    int sum(int r) {
        int ret = 0;
        for (; r >= 0; r = (r & (r + 1)) - 1)
            ret += bit[r];
        return ret;
    }

    int sum(int l, int r) {
        return sum(r) - sum(l - 1);
    }

    void add(int idx, int delta) {
        for (; idx < n; idx = idx | (idx + 1))
            bit[idx] += delta;
    }
};

BIT 2D

struct FenwickTree2D {
    vector<vector<int>> bit;
    int n, m;

    // init(...) { ... }

    int sum(int x, int y) {
        int ret = 0;
        for (int i = x; i >= 0; i = (i & (i + 1)) - 1)
            for (int j = y; j >= 0; j = (j & (j + 1)) - 1)
                ret += bit[i][j];
        return ret;
    }

    void add(int x, int y, int delta) {
        for (int i = x; i < n; i = i | (i + 1))
            for (int j = y; j < m; j = j | (j + 1))
                bit[i][j] += delta;
    }
};
