Construct the array

Posted 12/22/2022

Problem

Your goal is to find the number of ways to construct an array such that consecutive positions contain different values.

Specifically, we want to construct an array with nnn elements such that each element between 111 and kkk inclusive. We also want the first and last elements of the array to be 111 and xxx

Example

For example, for n=4,k=3,x=2n=4, k=3, x=2n=4,k=3,x=2 there are 333 ways, as shown here:

1 2 1 2
1 2 3 2
1 3 1 2

Constraints

  • 3≤n≤1053 \leq n \leq 10^53≤n≤105
  • 2≤k≤1052 \leq k \leq 10^52≤k≤105
  • 1≤x≤k1 \leq x \leq k1≤x≤k

Solution

Approach

We will apply the idea of Dynamic Programming.

  • Let f(i)f(i)f(i) indicates the number of ways to contruct array of iii length and the last number must not be xxx.
  • Let g(i)g(i)g(i) indicates the number of ways to contruct array of iii length and the last number must be xxx.

Easily we see the answer is g(n)g(n)g(n). We can calculate some initial value without much difficulty.

if x=1{f(1)=0f(2)=k−1g(1)=1g(2)=0}\text{if } x = 1 \left \{ \begin{align*} f(1) &= 0\\ f(2) &= k - 1\\ g(1) &= 1\\ g(2) &= 0\\ \end{align*} \right \}if x=1⎩⎨⎧​f(1)f(2)g(1)g(2)​=0=k−1=1=0​⎭⎬⎫​
 if x≠1{f(1)=1f(2)=k−2g(1)=0g(2)=1}\text{ if } x \neq 1 \left \{ \begin{align*} f(1) &= 1\\ f(2) &= k - 2\\ g(1) &= 0\\ g(2) &= 1\\ \end{align*} \right \} if x=1⎩⎨⎧​f(1)f(2)g(1)g(2)​=1=k−2=0=1​⎭⎬⎫​

From i=3i=3i=3 onward, the formular stays consistent regardless of xxx. f(i)f(i)f(i) and g(i)g(i)g(i) can be calculated using f(i−1)f(i-1)f(i−1) and g(i−1)g(i-1)g(i−1). There are only 1 way to pick xxx at position iii so

g(i)=f(i−1)g(i) = f(i - 1)g(i)=f(i−1)

If the previous number is not xxx we will lost 2 candidates. Thus, there are k−2k - 2k−2 ways to pick a number at iii If the previous number is xxx we only lost 1 candidate. Thus, there are k−1k - 1k−1 ways to pick a number at iii Combine them altogether we have

f(i)=(k−2)×f(i−1)+(k−1)×g(i−1)f(i) = (k - 2) \times f(i - 1) + (k - 1) \times g(i - 1)f(i)=(k−2)×f(i−1)+(k−1)×g(i−1)

That’s conclude our solution.

Code

#define INF 1000000007
long countArray(int n, int k, int x) {
    long f[100001];
    long g[100001];
    if(x == 1) {
        g[1] = 1;
        f[1] = 0;
    } else {
        g[1] = 0;
        f[1] = 1;
    }
    for(int i=2;i<=n;i++) {
        f[i] = ((k-2)*f[i-1] + (k-1)*g[i-1])%INF;
        g[i] = f[i-1];
    }
    return g[n];
}

You can optimize it further by only store latest fff and ggg, to get O(1)O(1)O(1) space complexity

#define INF 1000000007
long countArray(int n, int k, int x) {
    long f, g, tmp;
    if(x == 1) {
        g = 1;
        f = 0;
    } else {
        g = 0;
        f = 1;
    }
    for(int i=2;i<n;i++) {
        tmp = ((k-2)*f + (k-1)*g)%INF;
        g = f;
        f = tmp;
    }
    return f;
}
